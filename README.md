# Sistema Contas - Estágio

Sistema desenvolvido em Java com Spring para o setor administrativo, com o objetivo de gerenciar o cadastro e acompanhamento de contas de energia, água, telefonia e internet, facilitando o controle financeiro e a tomada de decisões.

## Sobre

O projeto visa criar um sistema completo de controle de contas para o setor administrativo, permitindo:

- Cadastro de contas de energia, água, telefonia e internet.
- Identificação de contas pendentes, pagas ou ainda não cadastradas.
- Realização de pesquisas por data de vencimento, inscrição, matrícula e outros parâmetros relevantes.

O desenvolvimento segue a Metodologia Ágil, garantindo ciclos curtos de entrega de valor, adaptação rápida a mudanças e colaboração contínua com o cliente. Além disso, o sistema conta com testes unitários para garantir a confiabilidade do software.

## Funcionalidades

- CRUD completo para contas de energia, água, telefonia e internet.
- Filtros e pesquisas por diferentes parâmetros (data de vencimento, inscrição, matrícula etc.).
- Registro de contas pendentes e pagas.
- Interface web desenvolvida com Spring Boot, facilitando o uso pelo setor administrativo.

## Tecnologias

- Java 
- Spring Boot
- Spring Data JPA
- Metodologia Ágil
